# US-14: Define Architecture in Rust with BWSK

**As** a Rust developer,
**I want** to define a neural network architecture using BWSK combinators in Rust and have it compile to optimized Burn modules with GPU support via CubeCL,
**So that** I get type-safe, high-performance ML in Rust.

## Acceptance Criteria

- [ ] BWSK DSL available as Rust API
- [ ] Compiles to Burn modules
- [ ] GPU execution via CubeCL
- [ ] Same classification results as Python implementation
